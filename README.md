# run_from_the_past
# Описание
В этой игре вам предстоит бежать... От самого себя! <br>
Ваши копии из прошлого будут идти по вашему следу... <br>
Так что не стойте на месте!

# Управление
Стрелки влево/вправо - движение
Стрелка наверх - прыжок
